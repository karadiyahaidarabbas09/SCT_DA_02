# SCT_DA_02

📊 HR Analytics Dashboard (Power BI)

This repository showcases an end-to-end HR Analytics project built using Power BI, designed to provide actionable insights into workforce trends, employee performance, and organizational metrics.

The dashboard enables data-driven decision-making by analyzing key HR indicators such as employee attrition, job satisfaction, performance ratings, and departmental distribution. It transforms raw HR data into interactive visualizations for better understanding and strategic planning.

🔍 Key Features
Employee Attrition Analysis
Department & Role-wise Insights
Job Satisfaction & Performance Metrics
Gender & Age Distribution Analysis
Interactive Filters and Drill-Down Reports
🛠️ Tools & Technologies
Power BI
Data Modeling
DAX (Data Analysis Expressions)
Data Cleaning & Transformation
🎯 Objective

To help organizations identify patterns in employee behavior, reduce attrition, and improve overall workforce efficiency through data visualization and analytics.
